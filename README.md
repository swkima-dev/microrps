# microrps

microrpsはRust製の自作TCP/IPプロトコルスタックです。
このプロジェクトは完全に学習目的であり、山本 雅也(やまもと まさや)氏の『ゼロからのTCP/IPプロトコルスタック自作入門』及び
koba789氏の『自作TCP/IPスタックをRustで実装するまで年を越せない【24時間耐久】』を参考にしています。

microrps is a self-made TCP/IP protocol stack written in Rust. 
This project is intended purely for learning purposes and is based on “ゼロからのTCP/IPプロトコルスタック自作入門” by Masaya Yamamoto 
and “自作TCP/IPスタックをRustで実装するまで年を越せない【24時間耐久】” by koba789.
